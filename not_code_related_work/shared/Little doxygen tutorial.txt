Doxygen - Install guide for windows

1. Download binary at
http://sourceforge.net/projects/doxygen/?source=typ_redirect
2. execute binary on windows
3. go thru gui-supported steps, then run applications
4. save folder to the Abgabe folder...

special commands in doxygen:
http://www.stack.nl/~dimitri/doxygen/manual/commands.html#

best commands for our purposes:
#to have the author's name
	@author
#to describe return values
	@retval <return value> {description} 
#prints "word" bold	
	@b <word>
#prints phrase bold
	<b>more than one word</b>	
#writes a brief description until next sectioning command or empty line
	@brief {brief description}
#describes a function-parameter
	@param<parameter-name>{parameter description}
#describes details of anything; either leave a blank line to start or use this command:
	@details{detailed description...}
